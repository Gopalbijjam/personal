# Gopal Portfolio

A Java full-stack portfolio website built with Spring Boot, backed by a small portfolio API and a responsive frontend that renders resume-driven content.

## Stack

- Java 21
- Spring Boot
- HTML, CSS, and JavaScript

## Run locally

```bash
mvn spring-boot:run
```

Open `http://localhost:8080`.

## What is included

- `GET /api/portfolio` to serve portfolio data from Java
- Responsive landing page tailored to Bijjam Gopal Reddy's resume
- Resume PDF available at `/assets/resume-gopi.pdf`
